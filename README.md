## 🧠 What This Tool Does

This calculator helps ecommerce sellers understand their true profitability by calculating:

- Product cost

- Shipping cost

- Ad spend per sale

- Platform fees (Shopify, Etsy, etc.)

- Final profit or loss per sale

It gives a **clear profit breakdown in seconds**.
